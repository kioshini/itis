# Contributing to Game Finder Bot

Спасибо за интерес к проекту! Мы приветствуем любой вклад.

## Как внести вклад

1. **Fork** репозитория
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`)
3. Внесите изменения и зафиксируйте их (`git commit -m 'Add some AmazingFeature'`)
4. Отправьте ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Стиль кода

- Следуйте PEP 8
- Используйте type hints
- Документируйте функции с помощью docstrings
- Пишите осмысленные commit messages

## Тестирование

Перед отправкой PR убедитесь, что:
- Код работает без ошибок
- Все функции протестированы
- Документация обновлена

## Сообщения об ошибках

При создании issue укажите:
- Версию Python
- Шаги для воспроизведения
- Ожидаемое поведение
- Фактическое поведение
- Логи ошибок
